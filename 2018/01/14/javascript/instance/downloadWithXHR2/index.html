XMLHttpRequest Level 2 实现文件下载

发表于 2018-01-14 | 分类于 javascript | 本文总阅读量 次

在 XMLHttpRequest level 2 中可以利用 Ajax 实现对二进制文件的下载功能，通过设置 responseType='blob'
